# LearnHub-FullStack
LearnHub - Full Stack Learning Management System
LearnHub is a comprehensive web application designed for managing educational content. This project is built using the MERN-equivalent stack, replacing MongoDB with MySQL for robust relational data management.
Tech StackLayerTechnology
Frontend - React.js, CSS3, JavaScript (ES6+)
BackendNode- .js, Express.jsDatabaseMySQL (Managed via XAMPP / MySQL Workbench)
